# Инструкция: Автоматический парсер кодов ТН ВЭД

Этот инструмент предназначен для автоматизации заполнения Excel-таблиц данными из таможенных деклараций (ГТД) и дополнений к ним.
## 1. Системные требования и установка
*   **ОС**: Windows (рекомендуется)
*   **Python**: Версия 3.8 или выше. Скачать можно на [python.org](https://www.python.org/).
*   **Зависимости**: Скрипт **автономный**. При первом запуске он автоматически скачает и установит необходимые библиотеки:
    *   `pdfplumber` (для чтения PDF)
    *   `openpyxl` (для работы с Excel)
    *   `pandas` (для обработки данных)

## 2. Как пользоваться инструментом
1.  Поместите скрипт `fill_tnved.py` и файл `ЗАПУСК_ПАРСЕРА.bat` в рабочую папку.
2.  Положите в эту же папку (или в ее подпапки) следующие файлы:
    *   **Таблица Excel**: Файл должен начинаться на «Табл...» (например, `Табл-1.xlsx`).
    *   **ГТД (PDF)**: Основной бланк декларации, имя начинается на «ГТД...».
    *   **Дополнение к ГТД (PDF)**: Файл с описанием моделей, имя начинается на «Доп...».
3.  Закройте файл Excel, если он открыт.
4.  Запустите **ЗАПУСК_ПАРСЕРА.bat**.

## 3. Требования к Excel-документу
Скрипт использует **динамический поиск колонок**, поэтому их точное положение не важно. Важны заголовки (в первых 10 строках):
*   **Наименование** (или «Модель»): Колонка, где указаны названия товаров. Скрипт ищет совпадение по первому слову/артикулу.
*   **№ товара** (или «Номер товара»): Колонка, куда будет записан порядковый номер из декларации. К этому номеру будет прикреплена **гиперссылка** на PDF-файл Дополнения.
*   **Код ТН ВЭД** (или «Код товара»): Колонка, куда будет записан 10-значный код ТН ВЭД.

## 4. Алгоритм поиска информации
Программа работает в три этапа:

### Этап А: Анализ Дополнения к ГТД
В Дополнении информация обычно разбита на блоки «Товар № X». Скрипт:
1. Разбивает PDF-текст на блоки по разделителю «Товар №».
2. Внутри каждого блока ищет ключевые слова (модели) из вашей Excel-таблицы.
3. Формирует связку: **Модель -> Номер товара**.

### Этап Б: Анализ основной ГТД
В основном бланке ГТД ищутся данные из граф 32 и 33. Скрипт:
1. Ищет в тексте паттерны вида `: <номер> <код>`.
2. Код ТН ВЭД должен состоять из 10 цифр.
3. Формирует связку: **Номер товара -> Код ТН ВЭД**.

### Этап В: Объединение и запись
1. Программа сопоставляет `Модель` из Excel с `Номером` (из Этапа А), а затем `Номер` с `Кодом` (из Этапа Б).
2. Обновляет ячейки в Excel, добавляя гиперссылку на PDF для быстрой проверки.
3. Сохраняет результат в новый файл с приставкой `_готово.xlsx`.
