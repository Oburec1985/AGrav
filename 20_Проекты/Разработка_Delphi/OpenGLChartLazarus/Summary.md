# OpenGLChartLazarus

Проект кроссплатформенного компонента графиков для Lazarus/FPC с OpenGL-отрисовкой.

На первом этапе ведется только проектная документация. Код компонента и пакет Lazarus будут начаты после фиксации архитектуры.

Код проекта планируется вести в `C:\Oburec\Projects\Lazarus`.

## Прообраз

Источник опыта: `C:\Oburec\OburecGH\sharedUtils\компоненты\chart_dpk\chart`.

Ключевой исходный класс старой версии: `uChart.pas`, `cChart = class(TPanel)`.

Важно: новая архитектура не строится по шаблону Delphi-компонента. Старый `cChart` нужен как источник сценариев и списка ошибок, которых надо избежать.
Пример возможностей: (несколько графиков, линии, курсоры, гистограммы, обработка пользовательских событий для каждого элемента (оси, линии, курсоры, подписи, логарифмирование осей) и т.д.)
