# TChartAxis

## Ответственность

Ось графика: ориентация, шкала, подписи делений и заголовок.

## Первые свойства

- ориентация: X или Y;
- диапазон;
- режим шкалы;
- форматирование подписей.

## Не делает

Ось не рисует себя сама. Она предоставляет рендеру данные для построения линий, сетки и подписей.
