# TJsonChartSerializer

## Ответственность

JSON-реализация [IChartSerializer](IChartSerializer.md).

## Назначение

Основной кандидат для первой версии формата: читаемый, удобен для секционного сохранения и кроссплатформенен.

## Что хранит

- версию схемы;
- float-layout объектов;
- страницы и их вложенные компоненты;
- viewport страниц;
- серии и стили;
- настройки активных режимов, если они нужны для восстановления рабочего состояния.

## Открытые вопросы

- Как именовать секцию графика.
- Нужна ли версия схемы в каждом сохраненном блоке.
- Как хранить пользовательские расширения.
