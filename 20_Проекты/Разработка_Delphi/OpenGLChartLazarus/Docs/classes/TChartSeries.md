# TChartSeries

## Ответственность

Базовый абстрактный класс серии данных.

## Общие свойства

- имя;
- видимость;
- стиль;
- расчет границ данных.

## Наследники

- [TLineSeries](TLineSeries.md) - первая реализация для линейного графика.

## Правило

Серия хранит данные и метаданные, но не содержит OpenGL-команды.
