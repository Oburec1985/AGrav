# TChartCursor

## Ответственность

Интерактивный курсор графика: вертикальный, горизонтальный или точечный маркер позиции.

## Содержит

- позицию в координатах графика;
- ориентацию;
- видимость;
- стиль отображения.

## Управляется

Пользовательское перемещение курсора выполняет [TCursorTool](TCursorTool.md). Сам курсор не обрабатывает мышь.
