# TChartAnnotation

## Ответственность

Базовый класс подписей, меток и пользовательских пометок на графике.

## Содержит

- текст;
- позицию;
- привязку к координатам данных или экрану;
- стиль.

## Наследники

Наследники будут добавлены позже: текстовая метка, диапазон, маркер события.
