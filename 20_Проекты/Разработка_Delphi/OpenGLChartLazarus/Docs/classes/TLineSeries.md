# TLineSeries

## Ответственность

Первая конкретная серия данных: линия по набору точек X/Y.

## Минимальный API

- добавить точку;
- очистить точки;
- получить количество точек;
- получить границы данных.

## Будущие вопросы

- Нужен ли буфер с заранее выделенной памятью для быстрых потоковых данных.
- Разделять ли обычную серию и ring-buffer серию.
- Как хранить пропуски данных.
