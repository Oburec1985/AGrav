# TChartStyleSet

## Ответственность

Набор стилей и палитр для модели графика.

## Содержит

- цвет фона;
- стиль сетки;
- палитру серий;
- стили осей, курсоров и подписей.

## Правило

Стиль описывает намерение, а рендер решает, как выразить его через OpenGL.
