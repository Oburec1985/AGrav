# Описание интерфейсов: ALF

## 1. Дизайн-концепция
- **Стиль**: Современный минимализм, "Glassmorphism" элементы, темная тема по умолчанию (Dark Mode first).
- **Акцент**: Карта и фотографии объектов.
- **Шрифт**: Inter или Outfit (без засечек).
## 2. Состав визуальных элементов

| Элемент | Назначение |
| :--- | :--- |
| **Header (Шапка)** | Логотип, глобальный поиск по ключевым словам, кнопка "+ Объявление" и смена языка. |
| **Map (Карта)** | Визуальное отображение всех активных меток. Позволяет искать вещи в конкретном районе и выбирать точку при создании объявления. |
| **Sidebar (Боковая панель)** | Фильтрация по категориям и датам, счетчик найденных объектов и лента последних находок (фон: Slate 800). |
| **Filters (Фильтры)** | Поиск в области видимости карты, выбор категорий и диапазона дат происшествия. |
| **Item Card (Карточка в списке)** | Краткий обзор: фото, тип (Потеряно/Найдено), дата, заголовок и адрес. |
| **Item Details (Панель просмотра)** | Подробная информация о вещи, галерея фотографий, отправка отклика и управление объявлением. |
| **Respond Form (Форма отклика)** | Позволяет нашедшему связаться с владельцем, прикрепить фото-доказательство и оставить свои контакты. |
| **Post Ad Modal (Создание)** | Пошаговая форма для публикации нового объявления с выбором локации на карте. |
| **Feedback Modal (Связь)** | Окно для связи с разработчиками и отправки предложений. |

## 3. Основные экраны (Web & Android)

### 2.1. Главный экран (Dashboard / Map)
- **Статистический бар** (сверху или снизу): "Найдено: 1250 | В поиске: 430 | Вернулось домой: 890".
- **Центральный элемент**: Интерактивная карта с метками.
- **Поиск**: Сверху плавающая строка поиска.
- **Фильтры**: Быстрые кнопки категорий (иконки: лапка, паспорт, смартфон).
- **Кнопка действия**: Крупная "+" кнопка "Сообщить о событии".

### 2.2. Форма создания объявления
- **Пошаговый процесс (Wizard)**: 
  1. Выбор типа (Потерял/Нашел).
  2. Выбор категории.
  3. Загрузка фото (Drag-and-drop / Камера).
  4. Карта для выбора точки.
  5. Описание и контакты.

### 2.3. Карточка объекта
- Галерея фото (слайдер).
- Статус: "В поиске", "На проверке", "Вернулся к владельцу".
- Кнопка "Это моё / Я видел это" (открывает чат).
- Таймлайн: когда создано, сколько просмотров.

### 2.4. Личный кабинет и Чаты
- **Персональная статистика**: "Вы помогли вернуть: 5 вещей | Ваша карма: +150".
- **Список своих объявлений**.
- **Мессенджер** для общения между нашедшим и потерявшим.
- **Уведомления** (колокольчик) о новых совпадениях от AI-свахи.

## 3. Адаптивность
- **Web**: Двухпанельный интерфейс (список слева, карта справа).
- **Android**: Bottom Navigation (Карта, Лента, Добавить, Чаты, Профиль). Плавные переходы, тактильная отдача (Haptic Feedback).
