# Проект Routerich: Обход DPI и Оптимизация сети

## Описание проекта
Основная задача — настройка роутера **RouteRich AX3000** для обеспечения стабильного доступа к ресурсам, замедляемым или блокируемым РКН (YouTube и др.), без внесения нестабильности в общую сеть.

## Ключевые достижения
- **DPI Bypass**: Реализовано через гибридную конфигурацию (Windows + Router).
- **Скрипты автоматизации**: Разработаны инструменты для настройки фаервола, смены DNS и проверки состояния обхода.
- **Rollback System**: Создана система экстренного отката настроек в случае потери связи.

## Важные файлы (в хранилище AGrav)
- [[emergency_rollback.py|Scripts/emergency_rollback.py]]: Скрипт экстренного восстановления связи.
- [[verify_hybrid.py|Scripts/verify_hybrid.py]]: Утилита проверки работы обхода.
- [[apply_tuned_v72.py|Scripts/apply_tuned_v72.py]]: Основной скрипт тюнинга параметров.

## Технические детали
- **Среда**: Double NAT (основной канал через вышестоящий роутер).
- **DNS**: Принудительная установка IPv4 DNS для предотвращения "утечек" через IPv6.
- **Фикс YouTube**: Применение специфических параметров TCP (TTL, Window Size) для обхода DPI.
