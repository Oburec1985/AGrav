# 🏗 Цех Проектов (Project Implementation)

Активные проекты, инструменты и реализации. Каждая папка содержит полную документацию (`Docs/`) и краткое резюме (`Summary.md`).

## 💻 Разработка ПО (Software Development)

### 🛰 [SQLSensorsDB](./SQLSensorsDB/Summary.md)
База данных датчиков (FastAPI + React). Продвинутый поиск, EAV-модель данных, импорт из Excel.

### 🌍 [Локализация Delphi (WPExtPack)](./Разработка_Delphi/Localization/localization_guide.md)
Перевод legacy-библиотек на английский язык, исправление проблем с Unicode и интеграция с GNU Gettext.

### 📈 [OpenGLChartLazarus](./Разработка_Delphi/OpenGLChartLazarus/Summary.md)
Кроссплатформенный OpenGL-компонент графиков для Lazarus/FPC. На первом этапе ведется документационное проектирование на базе Delphi-прототипа `cChart`.
