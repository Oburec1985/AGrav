# Настройка MTProxy для Telegram (mtg)

Инструкция по развертыванию прокси-сервера для Telegram с поддержкой Fake-TLS и защиты от цензуры.

---

## 🛠 Быстрая установка (Docker)

Если на сервере установлен Docker, выполните следующую команду (подставьте свой секрет):

```bash
docker run -d \
  --name mtg \
  --restart unless-stopped \
  -p 8888:8080 \
  cutenode/mtg run -b :8080 -s ee00112233445566778899aabbccddeeff636c6f7564666c6172652e636f6d -t cloudflare.com
```

---

## 📋 Таблица параметров (mtg)

| Параметр | Значение (Пример) | Описание и влияние |
| :--- | :--- | :--- |
| **Port** | `443` | Внешний порт для подключения. 443 — идеален, так как мимикрирует под HTTPS. |
| **Secret** | `ee...` | Секретный ключ. Префикс `ee` включает режим Fake-TLS, что критично для обхода DPI. |
| **Fake-TLS Domain** | `google.com` | Домен, под который маскируется рукопожатие (handshake). Должен совпадать с тем, что вшит в секрет. |
| **Bind Address** | `:8080` | Внутренний порт контейнера, который пробрасывается наружу. |
| **AD Tag** | `(пусто)` | Тег для продвижения канала. Если указан, ваш канал будет виден всем пользователям прокси сверху в списке чатов. |

---

## 🔑 Как сгенерировать правильный секрет
Используйте утилиту `mtg` для генерации или воспользуйтесь шаблоном:
1. Возьмите 32 случайных символа (hex).
2. Добавьте в начало `ee`.
3. Добавьте в конец hex-код домена (например, `google.com` в hex).

*Пример готовой ссылки для вставки в Telegram:*
`tg://proxy?server=138.124.85.38&port=8888&secret=ee00112233445566778899aabbccddeeff636c6f7564666c6172652e636f6d`

---

## 🛡 Безопасность и Firewall
Убедитесь, что выбранный порт (443) открыт в настройках вашего VPS (UFW/Iptables):
```bash
ufw allow 443/tcp
```
