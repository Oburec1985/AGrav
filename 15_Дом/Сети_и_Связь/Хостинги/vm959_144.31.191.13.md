# 🖥 Сервер vm959 (Германия, Александр)

Данный сервер используется для обхода блокировок через панель 3X-UI. Ниже приведено детальное описание настроенных соединений.

## 📋 Технические характеристики и доступ
- **IP**: `144.31.191.13`
- **Локация**: Германия
- **Панель управления**: [https://144.31.191.13:3295/HjD3gqhcq8RUdQw04J/](https://144.31.191.13:3295/HjD3gqhcq8RUdQw04J/)
- **Логин/Пароль**: См. [[Аккаунты]]

---

## 📊 Конфигурация соединений (Inbounds)

| Параметр | Значение | На что влияет |
| :--- | :--- | :--- |
| **Порт 443** | VLESS + Reality | **Основной канал.** Самый скрытный. Использует маскировку под сайт `museum.de`. Рекомендуется для постоянного использования. |
| **Порт 9443** | Trojan + gRPC | **Мобильный канал.** gRPC лучше работает при нестабильном сигнале (4G/LTE). |
| **Порт 10443** | VLESS + xHTTP | **Экспериментальный.** Имитирует стандартный HTTP-трафик для обхода жестких фильтров. |
| **Порт 11443** | VMess + WS | **Резервный.** Работает через WebSocket, что позволяет пробрасывать его через CDN. |

---

## ⚙️ Детальные параметры параметров (Reference)

| Параметр в панели | Конкретное значение | Влияние на соединение |
| :--- | :--- | :--- |
| **Protocol** | VLESS / Trojan / VMess | Тип протокола. VLESS — самый быстрый, Trojan — имитация HTTPS, VMess — классика. |
| **Security** | Reality | "Невидимость". Подменяет TLS-отпечаток сервера на отпечаток сайта `museum.de`. |
| **SNI / Dest** | `museum.de` | Сайт-прикрытие. Провайдер видит, что вы заходите именно сюда. |
| **Fingerprint (fp)** | `chrome` | Имитирует поведение браузера Chrome, чтобы избежать детектирования DPI. |
| **Reality PBK** | `YApjMm...` | Публичный ключ для авторизации клиента (Reality). |
| **Reality SID** | `1198c943...` | Короткий ID сессии для повышения уникальности. |
| **gRPC Service** | `grpc` | Имя сервиса для маршрутизации gRPC трафика. |
| **Path (WS/xHTTP)** | `/vmess`, `/xhttp` | Путь, по которому сервер понимает, что это VPN-запрос. |
