# 🦊 Как убрать окно авторизации прокси в Firefox (Работа)

Инструкция по настройке FoxyProxy для рабочего окружения (Mera), чтобы предотвратить появление окна авторизации при запуске.

---

## Шаг 1. Настройка FoxyProxy

1.  Нажать на иконку лисы -> **Настройки**.
2.  Нажать **Добавить**.
3.  Параметры подключения:
    *   **Название:** `Mera`
    *   **Тип:** `HTTP`
    *   **Адрес:** `prox.mera.local`
    *   **Порт:** `3128`
    *   **Логин:** `SkripnikAA`
    *   **Пароль:** `[ваш пароль]`
4.  Нажать **Сохранить**.
5.  В меню расширения выбрать: **"Использовать прокси Mera для всех URL"**.

## Шаг 2. Отключение конфликта в Firefox (ВАЖНО)

Чтобы браузер не выкидывал окно раньше расширения:

1.  Зайти в **Настройки Firefox** -> в поиске вбить "Прокси" -> кнопка **Настроить**.
2.  Выбрать пункт **"Без прокси"** и нажать ОК.
    > [!NOTE]
    > Теперь интернетом управляет расширение, в котором пароль уже забит.

## Шаг 3. Если окно всё равно вылезло (единоразово)

1.  Введите в адресной строке `about:config`.
2.  Найдите параметр `signon.autologin.proxy` и установите его в значение `true`.
