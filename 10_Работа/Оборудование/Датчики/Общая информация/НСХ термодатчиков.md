# 🌡️ Номинальные статические характеристики (НСХ) термодатчиков

Настоящий справочник содержит математические модели, коэффициенты полиномов и метрологические допуски для основных типов температурных датчиков (термосопротивлений и термопар) согласно российским и международным стандартам.

---

## 1. Термопреобразователи сопротивления (ТС / RTD)

Регламентируются стандартом **ГОСТ 6651-2009 (IEC 60751)**. Характеризуются сопротивлением при 0 °C ($R_0$) и температурным коэффициентом $\alpha$.

### 🔹 Прямое уравнение (Зависимость R от t)
Для платиновых термометров сопротивления (ПТС/Pt) используется уравнение Каллендара-ван Дюзена:

**Для диапазона от -200 °C до 0 °C:**
$$R_t = R_0 [1 + At + Bt^2 + C(t - 100)t^3]$$
**Для диапазона от 0 °C до 850 °C:**
$$R_t = R_0 (1 + At + Bt^2)$$
#### Коэффициенты (для $\alpha = 0.003851$ °C⁻¹):
| Параметр | Значение                                 |
| :------- | :--------------------------------------- |
| $A$      | $3.9083 \times 10^{-3} \text{ °C}^{-1}$  |
| $B$      | $-5.775 \times 10^{-7} \text{ °C}^{-2}$  |
| $C$      | $-4.183 \times 10^{-12} \text{ °C}^{-4}$ |

> [!NOTE]
> Для медных ТС (М) $\alpha = 0.00428$, зависимость линейна: $R_t = R_0(1 + 0.00428t)$.

### 🔹 Обратное уравнение (Зависимость t от R)
Для диапазона $t \ge 0 \text{ °C}$ расчет температуры по сопротивлению выполняется по квадратичной формуле:
$$t = \frac{-A + \sqrt{A^2 - 4B(1 - R_t/R_0)}}{2B}$$

## 2. Термоэлектрические преобразователи (Термопары)

Регламентируются стандартом **ГОСТ Р 8.585-2001 (IEC 60584-1)**. Характеризуются зависимостью ТЭДС ($E$, мВ) от разности температур горячего и холодного спаев.

### 🔹 Тип K (ТХА: Хромель-Алюмель)
Один из самых распространенных типов. Диапазон: -270...+1372 °C.

#### Обратная функция (мВ → °C) для диапазона 0...500 °C:
$$t = \sum_{i=0}^{n} d_i E^i$$

| Коэффициент | Значение     |
| :---------- | :----------- |
| $d_0$       | $0$          |
| $d_1$       | $25.08355$   |
| $d_2$       | $0.07860106$ |
| $d_3$       | $-0.2503131$ |
| $d_4$       | $0.0831527$  |
| $d_5$       | $-0.0122803$ |
| $d_6$       | $0.0009804$  |

## 3. Таблицы допусков (Tolerance Classes)

### Платиновые ТС (ГОСТ 6651)
| Класс допусков | Допуск, °C | Диапазон, °C |
| :--- | :--- | :--- |
| **AA** | $\pm(0.1 + 0.0017 \cdot |t|)$ | $-50 \dots +250$ |
| **A** | $\pm(0.15 + 0.002 \cdot |t|)$ | $-100 \dots +450$ |
| **B** | $\pm(0.3 + 0.005 \cdot |t|)$ | $-196 \dots +600$ |

### Термопары (ГОСТ Р 8.585) — пример для Типа K
| Класс | Допуск (что больше) | Диапазон, °C |
| :--- | :--- | :--- |
| **1** | $\pm 1.5 \text{ °C}$ или $\pm 0.4\% |t|$ | $-40 \dots +1000$ |
| **2** | $\pm 2.5 \text{ °C}$ или $\pm 0.75\% |t|$ | $-40 \dots +1200$ |

## 💻 Пример реализации (Python)

Ниже приведен код для расчета температуры Pt100 и термопары типа K.

```python
import math

class TemperatureSensor:
    # Коэффициенты Pt100 (ГОСТ 6651)
    PT100_R0 = 100.0
    PT100_A = 3.9083e-3
    PT100_B = -5.775e-7
    
    @classmethod
    def pt100_r_to_t(cls, r_measured):
        """Расчет температуры Pt100 по сопротивлению (для t >= 0)"""
        A, B, R0 = cls.PT100_A, cls.PT100_B, cls.PT100_R0
        discriminant = A**2 - 4*B*(1 - r_measured/R0)
        return (-A + math.sqrt(discriminant)) / (2 * B)

    @classmethod
    def type_k_e_to_t(cls, mv_measured):
        """Расчет температуры термопары типа K по ТЭДС (0...500 °C)"""
        # Коэффициенты NIST / ITS-90
        d = [0, 25.08355, 0.07860106, -0.2503131, 0.0831527, 
             -0.0122803, 0.0009804, -4.41303e-5, 1.057734e-6, -1.052755e-8]
        return sum(coeff * (mv_measured ** i) for i, coeff in enumerate(d))

# Тест:
print(f"Pt100 (138.5 Ohm): {TemperatureSensor.pt100_r_to_t(138.51):.2f} °C") # Ожидаем ~100
print(f"Тип K (4.096 mV):  {TemperatureSensor.type_k_e_to_t(4.096):.2f} °C")  # Ожидаем ~100
```

## 📄 Нормативная документация
1. **ГОСТ 6651-2009** — ГСИ. Термопреобразователи сопротивления из платины, меди и никеля.
2. **ГОСТ Р 8.585-2001** — ГСИ. Термопары. Номинальные статические характеристики преобразования.
3. **ITS-90** — International Temperature Scale of 1990.
