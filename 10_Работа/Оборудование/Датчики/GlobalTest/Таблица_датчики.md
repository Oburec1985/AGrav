Ограничения:
- нет встроенной поддержки объединения ячеек
- сложные таблицы лучше создавать в редакторах и вставлять ссылки и как изображение. Так же есть плагин **Database** для управления данными - проверить ег работу.

- Markdown не поддерживает фильтрацию, необходимо установить плагины:
    - **Dataview** - динамические таблицы на основе метаданных из  заметок
    - **DB Folder (Database)** -  превращает папки в базы данных с фильтрацией, сортировкой и поиском
    -  **Плагин Advanced Tables + Фильтрация вручную** - упрощенная фильтрация


- Работа с таблицами через:
     - google  - cоздайте таблицу в Google Sheets, опубликуйте ее и вставьте [[Вставка ссылок|ссылку]] в Obsidian.
     - excel - Экспорт таблицы в CSV → конвертирование в Markdown через [CSV to Markdown Table](https://tableconvert.com/csv-to-markdown).
