# Инструкция к парсеру GlobalTest

Этот набор скриптов предназначен для автоматического сбора данных о датчиках с сайта globaltest.ru и их организации в вашей базе Obsidian.

## Состав файлов:
1. **parse_globaltest.py** — Основной скрипт. Скачивает страницы, сжимает фото и создаёт заметки.
2. **restructure.py** — Группирует папки по категориям для чистоты структуры.
3. **processed_urls.txt** — История скачивания (чтобы не дублировать).

## Как пользоваться:
1. Установите Python и библиотеки: `pip install httpx beautifulsoup4 pillow lxml`
2. Запустите `python parse_globaltest.py` для скачивания новых данных.
3. Запустите `python restructure.py`, если нужно навести порядок в папках.

Данные сохраняются в `GlobalTest_Sensors`. На текущий момент обработано 794/796 товаров.
