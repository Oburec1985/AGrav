# Виброконтроллер AP5400

![[AP5400-GlobalTest.png]]

## Описание
### Назначение
Преобразование входного сигнала акселерометров с зарядовым выходом и акселерометров со встроенной электроникой.


Измерение виброскорости (СКЗ) объекта и выдача сигнала превышения заданного уровня вибрации в виде замкнутых или разомкнутых контактов реле, вывод результатов измерения на встроенный 4‑х разрядный индикатор, передача результатов измерения (СКЗ и форма сигнала) по интерфейсу RS-485 (CAN- опционально) и USB, вывод результатов измерения через унифицированный выход постоянного тока 4—20 мА
### Особенности
3 программируемых дискретных выхода (оптореле).


Режим индикации перегрузки.


Отображение величины измеряемой виброскорости (СКЗ) на встроенном индикаторе.


Изолированный унифицированный выход постоянного тока 4—20 мА для выдачи результатов измерений, возможность программирования 2—3,8 мА для выдачи дополнительных уставок.


Низкий уровень шумов.


Управление и настройка режимов работы через порт USB.
