# Усилитель  GT200A

![[GT200A_1.jpg]]

## Описание
### Назначение
Приём и усиление до нормированных значений сигналов преобразователей акустической эмиссии.
### Особенности
Работа на коаксиальную 50 Ом линию длиной до 100 м.
Малые собственные шумы.
Герметичный корпус.

## Технические характеристики
| Параметр | Значение |
| --- | --- |
| Коэффициент усиления | 1 … 200 (Определяется по требованию заказчика) |
| Пределы основной погрешности установки коэффициента усиления | ± 2 % |
| Диапазон рабочих частот с затуханием на границах минус 30% при длине линии |  |
| - 10 м | 10 … 800 кГц |
| - 50 м | 10 … 650 кГц |
| - 100 м | 10 … 400 кГц |
| Максимальное выходное напряжение (пик) при коэффициенте нелинейных искажений < 5%, не менее | ± (1 … 2) В |
| Входное сопротивление (более) | 100 кОм |
| Пределы допускаемой дополнительной относительной погрешности коэффициента преобразования по заряду в рабочем диапазоне температур - 40 … + 85 °С | ± 1 % |
