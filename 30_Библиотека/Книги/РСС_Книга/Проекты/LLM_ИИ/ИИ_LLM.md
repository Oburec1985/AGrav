**Уровни использования:**
1) В качестве чат бота. Вопрос ответ как wiki;
2) Инструменты для задание большого контекста - Cursor, Winserf, Gemini CLI и т.д. LLM не может сама по себе загрузить все файлы в память, в процессе размышления LLM подгружаеит документы частями, делает выводы и объединяет их. Давать большой контекст запросами не удобно, для повышения продуктивности необходимо, чтобы LLM могла работать с книгами, структурированными папками с документами.
3) Выполнение LLM задач, получение результатов. Дать возможность работы с инструментами (Excel, Google таблицы, PDF, построение видео и т.д., построение чертежей) и т.д.

**Пример работы с Cursor AI**
1) Разработка приложения для обработки изображений и видеопотоков;
2) Разработка оператора WinПОС

**Как установить:**
- Скачать с сайта;
- Настроить в браузере DNS или VPN. Например в браузере: Настройки>Безопасность и конфиденциальность>Дополнительные>Использлвать безопасный DNS сервер>Использовать персонализированные https://xbox-dns.ru/dns-query
- Так же появились готовые решения для роутеров с ОС openWRT для обхода геоблока;
- Запустить Cursor и выполнить подключение аккаунта. С 07.07.25 количество бесплатных запросов в месяц сильно убавилось (но по-прежнему можно насоздавать кучу аккаунтов (например, почтой Mail.ru) и переключаться между ними);
