# Тригонометрия для школьников

Введение: что такое тригонометрия и зачем она нужна в геометрии и физике.

## Основные функции

- $\sin\theta$ — синус угла $\theta$.
- $\cos\theta$ — косинус угла $\theta$.
- $\tan\theta = \dfrac{\sin\theta}{\cos\theta}$, для $\cos\theta \neq 0$.

## Основные тождества

- Пифагорова: $\sin^2\theta + \cos^2\theta = 1$.
- Обратные: $\csc\theta = \dfrac{1}{\sin\theta}$, $\sec\theta = \dfrac{1}{\cos\theta}$, $\cot\theta = \dfrac{\cos\theta}{\sin\theta}$.

## Формулы суммы и разности углов

- $\sin(\alpha \pm \beta) = \sin\alpha \cos\beta \pm \cos\alpha \sin\beta$.
- $\cos(\alpha \pm \beta) = \cos\alpha \cos\beta \mp \sin\alpha \sin\beta$.
- $\tan(\alpha \pm \beta) = \dfrac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha \tan\beta}$.

## Двойные и полувеличенные углы

- $\sin 2\theta = 2\sin\theta\cos\theta$.
- $\cos 2\theta = \cos^2\theta - \sin^2\theta$.
- $\tan 2\theta = \dfrac{2\tan\theta}{1-\tan^2\theta}$.
- Варианты для cos2: $\cos 2\theta = 2\cos^2\theta - 1 = 1 - 2\sin^2\theta$.

## Половинные углы

- $\sin \dfrac{\theta}{2} = \pm \sqrt{\dfrac{1 - \cos\theta}{2}}$.
- $\cos \dfrac{\theta}{2} = \pm \sqrt{\dfrac{1 + \cos\theta}{2}}$.
- $\tan \dfrac{\theta}{2} = \pm \sqrt{\dfrac{1 - \cos\theta}{1 + \cos\theta}} = \dfrac{\sin\theta}{1 + \cos\theta} = \dfrac{1 - \cos\theta}{\sin\theta}$.

## Примеры использования

- Пример 1: найти $\sin 30^\circ$, $\cos 30^\circ$, $\tan 30^\circ$.
- Пример 2: разложение $\sin(45^\circ + 15^\circ)$.

## Подборки по формулам
- Таблица базовых значений синуса и косинуса для частых углов: $0^\circ, 30^\circ, 45^\circ, 60^\circ, 90^\circ$.

## 3 гармонических сигнала с фазовым сдвигом 120°

Пусть сигналы:
- $s_1(t) = A_1 \\sin(\\omega t + a)$
- $s_2(t) = A_2 \\sin(\\omega t + a + 120^\\circ)$
- $s_3(t) = A_3 \\sin(\\omega t + a + 240^\\circ)$
Сумма: S(t) = s1(t) + s2(t) + s3(t).

Пусть комплексные фазоры:
- B = A1 e^{i a} + A2 e^{i(a+120^\circ)} + A3 e^{i(a+240^\circ)} = e^{i a} [A1 + A2 e^{i 120^\circ} + A3 e^{i 240^\circ}]
Тогда S(t) можно записать как
$$ S(t) = \operatorname{Im}\{ e^{i \omega t} e^{i a} B \} .$$

- Амплитуда суммарного сигнала: $R = |B|$.
- Фаза суммарного сигнала относительно базовой синусоиды: $\phi_S = a + \arg(B)$.
- Следовательно, искомая начальная фаза $a$ определяется как:
$$ a = \phi_S - \arg(B) \quad(\mathrm{mod}\ 2\pi). $$

- Упрощение через реальную и мнимую части:
  \( B = A1 + A2 e^{i 120^\circ} + A3 e^{i 240^\circ} = A1 - \frac{A2 + A3}{2} + i\frac{\sqrt{3}}{2}(A2 - A3) \)
- Тогда:
  \( R = \sqrt{\left(A1 - \frac{A2 + A3}{2}\right)^2 + \left(\frac{\sqrt{3}}{2}(A2 - A3)\right)^2} \)
- Арг(B): \( \arg(B) = \operatorname{atan2}\left( \frac{\sqrt{3}}{2}(A2 - A3), A1 - \frac{A2 + A3}{2} \right) \)

- Примеры расчета (числа):
- Пример 1: A1=5, A2=5, A3=0, φ_S = 0°.
- B = 5 + 5 e^{i 120°} = 2.5 + i 4.3301
- |B| = 5
- arg(B) ≈ 60°
- a = φ_S - arg(B) = -60° ≡ 300°

- Пример 2: A1 = 2, A2 = 3, A3 = 1, φ_S = 30°.
- Real(B) = A1 − (A2 + A3)/2 = 0
- Imag(B) = (√3/2)(A2 − A3) = √3 ≈ 1.732
- B ≈ 0 + i 1.732
- |B| ≈ 1.732
- arg(B) ≈ 90°
- a = φ_S − arg(B) = -60° ≡ 300°

- Пример 3: A1=A2=A3=4.
- B = 4 + 4 e^{i120°} + 4 e^{i240°} = 0
- R = 0
- Фаза не определена.

Примечания:
- Если A1 = A2 = A3, то B = 0 и сумма равна нулю.
- Зависимость фазы от разности A2 и A3.

## Расчет фазы a для трёх гармоник (a, a+120°, a+240°)

Пусть сигналы:
- $s_1(t) = A_1 \sin(\omega t + a)$
- $s_2(t) = A_2 \sin(\omega t + a + 120^\circ)$
- $s_3(t) = A_3 \sin(\omega t + a + 240^\circ)$
Сумма: $S(t) = s_1(t) + s_2(t) + s_3(t)$.

Пусть комплексные фазоры:
- $B = A_1 e^{i a} + A_2 e^{i(a+120^\circ)} + A_3 e^{i(a+240^\circ)} = e^{i a} [A_1 + A_2 e^{i 120^\circ} + A_3 e^{i 240^\circ}]$
- Амплитуда: $R = |B|$.
- Фаза суммарного сигнала: $\phi_S = a + \arg(B)$.
- Определение: $a = \phi_S - \arg(B) \; (\mathrm{mod}\; 2\pi)$.

- Упрощение через реальную и мнимую части:
  $B = A_1 - \frac{A_2 + A_3}{2} + i \frac{\sqrt{3}}{2} (A_2 - A_3)$
- Тогда:
  $R = \sqrt{\left(A_1 - \frac{A_2 + A_3}{2}\right)^2 + \left(\frac{\sqrt{3}}{2}(A_2 - A_3)\right)^2}$
- Арг(B): $\arg(B) = \operatorname{atan2}\left( \frac{\sqrt{3}}{2}(A_2 - A_3), A_1 - \frac{A_2 + A_3}{2} \right)$

Примеры использования:
- Пример 1: A1=5, A2=5, A3=0, φ_S = 0. Найдите B, |B| и arg(B), затем a = φ_S - arg(B).
- Пример 2: A1=2, A2=3, A3=1, φ_S = 30°.
- Пример 3: A1=A2=A3=4. В этом случае B=0.
