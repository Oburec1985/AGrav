# История изменений

Лог значимых изменений во всех проектах.

## 2026-04-23
- **VLESS+Reality**: Настройка Reality (порт 4433) + WARP Outbound на hip.hosting.
- **Инфраструктура**: Обновлена инструкция [Настройка VLESS Reality](../15_Дом/Сети_и_Связь/40_Энциклопедия_Обхода/12_Настройка_VLESS_Reality.md).
- **Система**: Введены правила логирования, описания параметров и структуры инструментов в [Скиллы ИИ](../00_Система/Скиллы_ИИ.md).
- **Инструментарий**: Реорганизация `Инструменты` (MD + Папка). Создан аудит-скрипт `kb_maintenance.py`.
- **Восстановление**: База успешно восстановлена через `git checkout .` после сбоя массовой правки.
- **Гигиена**: Глобальная чистка хранилища:
    - Перенос orphan-изображений из корня в `00_Система/Attachments/`.
    - Удаление дубликатов заметок в разделе `30_Библиотека`.
    - Массовая замена `[[Навигация.md]]` -> `[[Оглавление]]`.
    - Стандартизация именования (удаление пробелов в системных путях).
- **Навигация**: Обновлено [Оглавление](../00_Система/Оглавление.md), добавлены ссылки на Лог изменений и Аккаунты.
- **Протокол Отладки**: Внедрен стандарт **High-Res Logging** (TID, миллисекунды, авто-очистка) для повышения автономности ИИ при поиске багов в коде ([Код Стандарты](../00_Система/Код_Стандарты.md)).
- **MTProxy**: Развернут и задокументирован прокси для Telegram (порт 8888, Fake-TLS cloudflare.com) без использования Docker Hub. Инструкция: [Настройка MTProxy](../15_Дом/Сети_и_Связь/40_Энциклопедия_Обхода/13_Настройка_MTProxy.md).
- **Траблшутинг**: Восстановлена работа `oburec-in` (исправлен битый ключ и включен `No Kernel Tun` для Cloudflare WARP в панели 3X-UI).

## 2026-04-12
- Реорганизация хранилища знаний AGrav.
- Дистилляция глобального манифеста АИ в специализированные заметки.
- Восстановление истории и архитектуры проекта **SQLSensorsDB** (Парсер Obsidian, База данных датчиков).
- Добавлены инструкции по запуску и развертыванию SQLSensorsDB (Backend, Frontend, Importer).

## 2026-04-13
- **Оптимизация роутера**: Проведена чистка RouteRich AX3000 (кэш opkg, LuCI, логи).
- **Реструктуризация Доменов**: Создан домен `15_Дом` для отделения личных сетевых настроек от профессиональной экспертизы.
- **Оптимизация трафика**: Telegram Desktop, WhatsApp (Meta) переведены в VPN-канал (`awg10`) для ускорения загрузки медиа.
- **Дебаг 4pda**: Выявлен конфликт с `zapret` и скрытые IPv6 утечки. Применена блокировка IPv6 (AAAA) через `dnsmasq`. Сайт 4pda пока оставлен на внешнем VPN по желанию пользователя.
- **Стабилизация**: Проведен селективный откат настроек (Dont Touch My DHCP!), восстановивший работу Telegram Web.
- **Decommissioning Zapret**: Сервис `zapret` (nfqws) отключен как нестабильный и "вешающий" сеть.
- **Обновление Skills ИИ**: Введены правила **KB Infinity** (полнота логов) и **Increment & Snapshot** (обязательные бэкапы перед опытами на роутере).
- Подробные инструкции добавлены в [[Обслуживание_RouteRich 192.168.2.2|Обслуживание RouteRich]].
