# ⚙️ Инструментарий AGrav (Tooling)

Центральный реестр автоматизированных инструментов для обслуживания и развития базы знаний.

---

## 🛠 Активные инструменты

### 📚 [Парсинг Книг](./Инструменты/Парсинг_Книг.md)
Автоматизация извлечения текста и метаданных из PDF/EPUB.
- **Скрипты**: `parse_pdf.py`, `smart_parse_pdf.py`.
- **Назначение**: Формирование библиотеки.

### 🌐 [Перевод MD](./Инструменты/Перевод_MD.md)
Пакетный перевод markdown-файлов с сохранением структуры.
- **Скрипты**: `smart_translate_md.py`.
- **Назначение**: Локализация документации.

### 🧹 [Сборка Мусора](./Инструменты/Сборка_Мусора.md)
Аудит целостности базы данных.
- **Скрипты**: `kb_maintenance.py`.
- **Назначение**: Поиск битых ссылок и дубликатов (Audit Only).

---

## 📋 Правила создания инструментов
- **MD + Folder**: У каждого инструмента должен быть `.md` файл с инструкцией и одноименная папка со скриптами.
- **Параметры**: В инструкции обязательно указывать описание параметров запуска.
- **Безопасность**: Скрипты обслуживания по умолчанию работают в режиме `Read-Only Audit`.
