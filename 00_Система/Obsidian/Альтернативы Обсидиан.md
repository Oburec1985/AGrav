- Нет встроенной синхронизации в реальном времени.
   - Нет блокировок файлов (риск конфликтов: двое редактируют один файл → изменения одного перезапишут другого) - как выяснилось, это проблема многих программ, в Notion это даже более проблемно (может стереть данные без предупреждения), чем в Обсидиан.

- **Микро-команды (2-3 человека)** с железной дисциплиной:
       - Чёткие правила: «Редактируем только свои файлы», «Перед открытием хранилища — принудительная синхронизация».
       - Использование **Git** с код-ревью через GitHub/GitLab.
       - Например, документацию пишет один, остальные только читают через Obsidian Publish или экспорт в PDF.

### Logseq vs Obsidian

| **Критерий**             | **Logseq**                                                         | **Obsidian**                                                                                           |
| ------------------------ | ------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------ |
| **Структура**            | Блоки (outliner)                                                   | Плоские заметки (MD-файлы)                                                                             |
| **Ссылки**               | На страницы + **на блоки**                                         | Только на страницы                                                                                     |
| **Daily Notes**          | Центральный элемент                                                | Один из плагинов                                                                                       |
| **PDF-аннотации**        | Встроенные + цитирование                                           | Через плагины (e.g., Annotator)                                                                        |
| **Мультимедиа**          | Whiteboards (визуальные канвасы)                                   | Через плагины (Excalidraw)                                                                             |
| **Коллаборация**         | Есть (Whiteboards + Sync)                                          | Нет (только ручная настройка, бэкап данных)                                                            |
| **Мобильное приложение** | Стабильное (iOS/Android)                                           | Официальное (но слабее десктопа)                                                                       |
| **Текст**                | Может не вывозить (нуждается в тестировании) большое кол-во данных | Для большого кол-ва текста                                                                             |
| **Таблицы**              | Тест                                                               | С помощью плагинов, пока не удалось достичь желаемого результата (фильтры, масштабирование, изменение) |
| **Синхронизация в git**  | Тест на локальном компьютере, на рабочем не получается             | Тест на локальном компьютере, на рабочем не получается                                                 |
