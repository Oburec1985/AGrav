**Действия**

| **Действие<br>**               | **Клавиши** (англ) |
| ------------------------------ | ------------------ |
| Создать новую заметку          | Ctrl + N           |
| Сохранить Заметку              | Ctrl + S           |
| Поиск по всем заметкам         | Ctrl + Shift + F   |
| Быстрый поиск                  | Ctrl + O           |
| Командная палитра              | Ctrl + P           |
| Граф связей                    | Ctrl + G           |
| Режим просмотра/редактирования | Ctrl + E           |

**Редактор текста**

| Действие | Клавиши  |
| -------- | -------- |
| Жирный   | Ctrl + B |
| Курсив   | Ctrl + I |
| Ссылка   | Ctrl + K |

Можно так же назначить свои горячие клавиши **в настройках** и отредактировать для общего удобства пользования
