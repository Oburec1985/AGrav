# Инструменты и MCP Серверы

Специфические инструменты для расширения возможностей ИИ.

## Работа с Excel
- **ExcelMcpPrj.exe**: Скомпилированный MCP-сервер для работы с Excel.
- **Принцип работы**: Отличается от классических MCP. 
- **Инструкции**: Перед использованием изучи `tools_ru.md` в корне проекта или соответствующей папке.

## Прочие серверы
- **Context7**: Сервер для получения актуальной документации по библиотекам и фреймворкам. 
    - *Статус*: В процессе подключения.
