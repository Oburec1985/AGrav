# 🧹 Инструмент: Сборка Мусора (KB Maintenance)

Инструмент для поддержания целостности базы знаний AGrav. Позволяет находить дубликаты файлов и битые ссылки без риска автоматического удаления.

---

## 🛠 Как пользоваться (Manual)

### 1. Запуск аудита
Для генерации отчета выполните команду:
```powershell
python 00_Система/Инструменты/Сборка_Мусора/kb_maintenance.py
```
Результат будет сохранен в: `scratch/audit_report.txt`

### 2. Чтение отчета
- **BROKEN LINKS**: Список файлов и ссылок внутри них, которые ведут "в никуда".
- **DUPLICATES**: Группировка файлов с идентичным содержимым (по хэшу MD5).

### 3. Алгоритм безопасной чистки
1.  **Анализ дублей**:
    - Найдите группу дубликатов в отчете.
    - Выберите "Победителя" (Winner) — файл, который находится в более логичном месте (например, в `10_Работа` вместо `90_Архив`).
    - Удалите остальные файлы вручную или через команду `rm`.
2.  **Исправление ссылок**:
    - Перейдите в файл, указанный в отчете как источник битой ссылки.
    - Исправьте путь или название файла.
    - Если файл был удален как дубликат — обновите ссылку на "Победителя".

---

## 📋 Технические детали
- **Скрипт**: `kb_maintenance.py`
- **Режим**: `Audit Only` (только чтение). Скрипт **никогда** не удаляет файлы сам.
- **Хэширование**: Используется MD5 для точного сравнения содержимого, независимо от имени файла.
