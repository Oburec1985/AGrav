# Заметки по проекту Python_Excel

## Дата: 27.04.2026
**Задача**: Исправить ошибку `Error opening data file .../rus.traineddata` при запуске `parse_zp_image.py`.

**Анализ**:
- В системе установлен Tesseract OCR, но только с английским языком.
- Скрипт требует `lang='rus'`.
- Прямая запись в `C:\Program Files` может требовать прав администратора.

**Решение**:
- Добавлена функция `ensure_rus_language`.
- Если языка нет, он скачивается в `./tessdata/rus.traineddata`.
- В `pytesseract` передается конфиг `--tessdata-dir` указывающий на локальную папку.

**Статус**: Выполнено. Файл скачан, скрипт готов к работе.
